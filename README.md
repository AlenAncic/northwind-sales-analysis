# Northwind Traders Sales Analysis

## Project Overview
Analysis of Northwind Traders sales data covering the period 1996-1998, 
including 830 orders, 91 customers and 77 products across 8 categories.

## Tools Used
- PostgreSQL
- DBeaver
- Power BI

## Key Findings
- Total revenue: $1,265,793
- Beverages is the top revenue category with $267,868
- Top 3 customers generate 25% of total revenue
- Margaret Peacock leads employee performance with 18% of total revenue
- Revenue shows consistent upward trend throughout the period

## Files
- `northwind_analysis.sql` — SQL queries and analysis
- `views.sql` — PostgreSQL views for Power BI
- `northwind_dashboard.pdf` — Power BI dashboard

## Dataset
Northwind Traders — fictional food import/export company database
